# QuickStart 快速起步

准备好使用 HooperUI 了吗？在此之前你最好掌握下方技能来更加灵活的使用：

* [x] Javascript、ES6 and Vue3.0
* [x] CSS、SASS
* [x] HTML5
* [x] 模块化、ES module

!!! tip
    不掌握上方技能，你也可以参照我们的文档来使用 HooperUI，但我们仍然建议你详细的学习和理解上方技能，便于你深入理解和学习 HooperUI 的运作机理。

## npm 引入

推荐使用 npm 的方式安装，它能更好地和 webpack 打包工具配合使用。

``` bash
npm i hooperui -S
```
